## Clickhouse Cluster
ClickHouse Cluster Setup with 1 Shard and 3 Nodes

### docker command

``` bash
# start service
$ docker compose up -d

# shut down servce 
$ docker compose down 


# curl below url to check
curl http://localhost:8123/ping
# return "Ok."

```
